# HTML5 设计目的

HTML5的设计目的，是在移动设备上支持多媒体

  • 引入新的多媒体元素： 如video 、 audio；
  • 引入图形绘制： canvas
  • 引入新的语义结构元素： footer、article、aside、nav等
  • 引进新的表单控件：calendar、date 、time、 email、url、search等

# HTML5新增的特性

  • 新元素
  • 新属性
  • 支持CSS3
  • 2D/3D制图
  • 本地存储
  • Web应用
  • Video支持
  • Audio支持

# HTML5 新增标签

  • header（网站头部）
  • nav （导航栏）
  • section（类似div）
  • aside（文件侧栏）
  • article（文章/文本内容）
