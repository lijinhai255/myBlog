# 微信小程序目录结构

# 总览

pages: 文件夹 该文件夹用于盛放页面
utils: 文件夹 用于盛放工具JS文件
app.js: 应用程序构造器脚本
app.json: 应用程序配置文档
app.wxss: 全局样式 
project.confi.json: 项目配置文档

# pages

首先，该文件夹下是所有的子文件夹，每一个子文件夹，表示一个页面。
其次，pages这个名字，可以是任意的。

# utils

utils是一个工具文件夹。里面存放的是需要用到的所有工具。
